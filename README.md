# 🛒 E-Commerce Shopping Cart System (Java)

## 📌 Project Overview

This project is a simple **E-Commerce Shopping Cart System** developed using **Core Java and Object-Oriented Programming (OOP)** concepts.
It allows users to add, remove, and view products in a shopping cart and generates a final bill with discount and tax.

---

## 🚀 Features

* Add products to cart
* Remove products from cart
* View cart items
* Calculate total price dynamically
* Apply discount (10%) and tax (5%)
* Generate final bill at checkout

---

## 🧠 OOP Concepts Used

* Classes and Objects
* Encapsulation
* Modular Design
* Method Implementation

---

## 🏗️ Project Structure

* **Product Class** → Stores product details (ID, name, price, quantity)
* **Cart Class** → Handles cart operations (add, remove, display, total calculation)
* **Main Class** → Menu-driven program for user interaction

---

## 💻 Technologies Used

* Java (JDK 8 or above)
* VS Code / Eclipse

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/ecommerce-cart-system.git
```

2. Navigate to project folder:

```
cd ecommerce-cart-system
```

3. Compile the program:

```
javac ECommerceShoppingCartSystem.java
```

4. Run the program:

```
java ECommerceShoppingCartSystem
```
